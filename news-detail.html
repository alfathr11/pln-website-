PLN Luncurkan Program Diskon Listrik

Tanggal: 10 Mei 2025

PLN memperkenalkan token listrik dengan harga khusus untuk pelanggan prabayar sebagai bagian dari program diskon bulan ini. Program ini bertujuan untuk membantu masyarakat dalam menghemat biaya listrik dan meningkatkan kenyamanan penggunaan listrik.

Program diskon ini berlaku untuk pelanggan prabayar di seluruh wilayah Indonesia dan dapat diakses melalui aplikasi resmi PLN serta outlet token listrik resmi.
